# Long Term Planner

![planning](https://img.shields.io/badge/planning-blue?style=flat-square) ![productivity](https://img.shields.io/badge/productivity-blue?style=flat-square) ![deer-flow](https://img.shields.io/badge/deer--flow-blue?style=flat-square)

Agent specializing in breaking down yearly goals into monthly and weekly tasks.

## Overview
Long Term Planner is a specialized Python agent designed to transform ambitious yearly objectives into manageable, actionable steps. By leveraging structured decomposition, it bridges the gap between high-level vision and day-to-day execution. It ensures that every weekly task aligns with your monthly milestones and long-term targets.

## Features
*   **Hierarchical Decomposition:** Automatically breaks down a single yearly goal into 12 monthly milestones.
*   **Actionable Weekly Tasks:** Generates specific, high-impact tasks for every week of the year.
*   **Context Awareness:** Adjusts complexity based on the scope and timeline of the provided objective.
*   **Deer-Flow Integration:** Optimized for seamless workflow management within the deer-flow ecosystem.

## Installation
Ensure you have Python 3.8+ installed. Clone the repository and install the dependencies:

```bash
git clone https://github.com/your-username/long-term-planner.git
cd long-term-planner
pip install -r requirements.txt
```

## Usage
To start the planning agent, run the main script and follow the prompts to input your goal.

```bash
python main.py
```

**Example:**
If your goal is "Learn Data Science and build a portfolio," the agent will generate:
- **January:** Master Python fundamentals and NumPy.
- **Week 1:** Complete basic syntax tutorials and set up environment.
- **Week 2:** Practice list comprehensions and data structures.
